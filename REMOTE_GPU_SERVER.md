但是由于这个机器的网络问题，我们难以直接在上面运行 VSCODE SERVER。所以为了再这上面进行试验，我们要使用 ssh 命令在上面执行命令并获取反馈。例如：

```bash
ykdz@ASUS-TUF-Gaming:~$ ssh 4090 "echo yes"
yes
```

如果想要修改某些文件，让我们也通过 ssh 进行。你可以先在本地修改好文件，然后将文件上传到远端对应目录下。

在对方的环境中，代码被克隆到 `/data/home/ykdz/FedML` 目录下。目前的版本与本地版本完全同步。
